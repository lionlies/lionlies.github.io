My Gem Set

Posted on 2016-09-22   |  

當初是看了這兩篇，想說也來列個自己常用的 gem list, 開發新專案時直接複製貼上就好，非常方便。
http://felixclack.com/post/90434933071/what-are-my-goto-set-of-gems
http://kakas-blog.logdown.com/posts/364933-gem-finishing

後來在 Ruby China 上看到有人把常用的 gems 裝好後打包成一個 Rails 模板，之後可以來試試看。
https://ruby-china.org/topics/30406

必裝

gem “quiet_assets”
gem “better_errors”
gem “awesome_rails_console”
gem “guard-livereload”, group: :development
gem “rails-erd”
gem “meta_request”

差不多是必裝

gem “font-awesome-rails”
gem “simple_form”
gem “bootstrap-sass”
gem “bullet”
gem “devise”

Gemfile

需要上傳圖片

gem “carrierwave”
gem “mini_magick”

brew install imagemagick # 另外有一個 “image_magick” gem, 但不是我們要的。
http://git.imagemagick.org/repos/ImageMagick

需要 deploy 到 heroku

gem 介紹

rails-bootstrap-forms

https://github.com/bootstrap-ruby/rails-bootstrap-forms

bullet

help to kill N+1 queries and unused eager loading
https://github.com/flyerhzm/bullet

friendly_id

https://github.com/norman/friendly_id

devise

https://github.com/plataformatec/devise
https://github.com/plataformatec/devise#controller-filters-and-helpers

$ rails generate devise:install
$ rails generate devise model_name

figaro

nokogiri

http://www.nokogiri.org/

mechanize

https://github.com/sparklemotion/mechanize

pry

debug用

aasm

letter_opener

roadie

country_select

https://github.com/stefanpenner/country_select

kaminari

做 index 分頁（每個分頁有 N 筆資料）
https://github.com/amatsuda/kaminari
https://ihower.tw/rails4/basic.html#section-7

haml-rails

以 haml 取代 erb
相較於ERB，使用 HAML 雖然可以更為有效率地撰寫 HTML 樣板，但是還是需要考量團隊中的網頁設計師是否能夠配合使用。
http://haml.info/

jQuery plugin

gridalicious

http://suprb.com/apps/gridalicious/
https://github.com/suprb/Grid-A-Licious
要去 github 下載它的 .js 檔 jquery.grid-a-licious.js 或 jquery.grid-a-licious.min.js

類似的有 Masonry
