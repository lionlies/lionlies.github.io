限制表單輸入的字數

Posted on 2016-09-19   |  

可以從兩個方面去設定：

  1. HTML
  2. Model validation

HTML 的部分，在 posts/new.html.erb 中

我們可以透過 input_html: {maxlength: 30} 限制只能輸入30個字，多的沒辦法再輸進表單。

Model validation 的部份，顧名思義就是在 Post model 設一個 validation 限制輸入多少個字，比如 30, 表單內輸入的長度可以超過30個字，但是會無法儲存。
